## 2. 分析总体原则

- 所有主要分析均按预注册Protocol执行。
- 所有效应量在合并前转换至适当尺度，如log RR、log OR、log HR、Fisher z。
- 主要结论基于主要结局、主模型、偏倚风险和GRADE综合判断。
- 探索性分析与预设分析分开报告。

## 3. 数据准备

### 3.1 数据检查

- [ ] 检查重复研究。

### 3.2 数据转换规则

| 数据类型 | 转换规则 |
|---|---|
| RR/OR/HR | 取自然对数，SE = (log upper - log lower)/(2×1.96) |
| 连续型 | 优先MD；不同量表使用SMD |
| 比例 | logit转换或GLMM |
| 相关系数 | Fisher z转换 |
| 诊断四格表 | TP、FP、FN、TN录入 |
| 剂量反应 | 统一剂量单位，估计分组中点 |

## 9. 敏感性分析

| 敏感性分析 | 目的 | 预期解释 |
|---|---|---|
| 排除高偏倚风险研究 | 检查偏倚影响 | 若结果改变，降低证据确定性 |
| 改变τ²估计方法 | 检查模型依赖性 | 若结果改变，说明模型不稳健 |
| 固定/随机模型对比 | 检查模型假设影响 | 不作为机械选择依据 |
| Leave-one-out | 检查单项研究影响 | 识别高影响研究 |
| 排除转换数据 | 检查估算数据影响 | 判断数据转换可靠性 |
| 不同零事件处理 | 罕见事件稳健性 | 安全性结局重点 |

## 10. 发表偏倚与小样本效应

若k≥10：

- Funnel plot。
- Egger或适当替代检验。
- Trim-and-fill作为探索性。

若k<10：

- 不进行正式检验或仅作为描述。
- 在讨论中说明无法可靠评价。

## 11. 特殊类型分析计划

### 11.1 诊断准确性Meta

- 使用双变量随机效应模型或HSROC。
- 报告敏感度、特异度、LR+、LR-、DOR、SROC。
- 评价阈值效应。
- 使用QUADAS-2。

### 11.2 网状Meta

- 评价transitivity。
- 绘制network plot。
- 评价局部和全局不一致性。
- 报告league table和排序不确定性。
- 排名不作为唯一结论。

### 11.3 剂量反应Meta

- 统一剂量单位。
- 估计开放区间中点。
- 检验线性与非线性。
- 使用限制性立方样条。

### 11.4 预测模型Meta

- 开发、内部验证、外部验证分开。
- AUC/C-index与校准指标分开。
- 使用PROBAST/PROBAST+AI。
- 不合并高度异质或报告不足的性能指标。

## 12. 多重性处理

- 明确主要结局和主要模型。
- 次要和探索性分析不用于强结论。
- 不因P值筛选结果。
- 在讨论中说明多重比较风险。

## 13. 缺失数据处理

- 联系作者获取缺失数据。
- 使用既定公式换算SE/SD。
- 图形重建数据标记为估算。
- 对估算数据进行敏感性分析。

## 15. 结论规则

结论根据以下综合判断：

1. 主分析效应量和CI。
2. 临床重要性。
3. 预测区间。
4. 偏倚风险。
5. 异质性和稳健性。
6. 发表偏倚。
7. GRADE证据确定性。

低或极低确定性证据不得写成确定性临床建议。
