# 04｜偏倚风险、报告规范与证据确定性分级

## 1. 三个概念必须分开

| 概念 | 关注点 | 典型工具 |
|---|---|---|
| 报告质量 | 论文是否把该报告的信息写清楚 | PRISMA、CONSORT、STROBE、TRIPOD |
| 偏倚风险 | 研究结果是否系统性偏离真实值 | RoB 2、ROBINS-I、QUADAS-2、PROBAST |
| 证据确定性 | 整体证据能否支持结论 | GRADE、CINeMA、证据可信度分级 |

常见错误：用NOS、Jadad或“质量评分”替代偏倚风险；用PRISMA评分替代方法质量；用P值替代证据确定性。

## 2. 报告规范选择

| 论文/研究类型 | 报告规范 |
|---|---|
| 系统综述/Meta分析 | PRISMA 2020 |
| Protocol | PRISMA-P |
| 检索策略 | PRISMA-S |
| 诊断准确性系统综述 | PRISMA-DTA |
| 网状Meta | PRISMA-NMA |
| IPD-MA | PRISMA-IPD |
| 范围综述 | PRISMA-ScR |
| 无法Meta的干预综述 | SWiM |
| 预测模型系统综述/Meta | TRIPOD-SRMA |
| AI预测模型原始研究 | TRIPOD+AI |
| RCT原始研究 | CONSORT |
| 观察性研究 | STROBE |
| 诊断准确性原始研究 | STARD |

## 3. 偏倚风险工具

## 3.1 RoB 2：随机对照试验

适用于个体随机RCT；也有cluster RCT和crossover RCT版本。

### 主要领域

1. 随机化过程偏倚。
2. 偏离预定干预的偏倚。
3. 缺失结局数据偏倚。
4. 结局测量偏倚。
5. 选择性报告结果偏倚。

### 关键点

- RoB 2是结局层面评价，而不是整篇研究一个评分。
- 不同结局的偏倚风险可能不同。
- 盲法的重要性取决于结局是否主观。
- ITT、per-protocol、as-treated对应不同estimand。

## 3.2 ROBINS-I：非随机干预研究

适用于非随机研究评价干预效果。

### 主要领域

1. 混杂偏倚。
2. 研究对象选择偏倚。
3. 干预分类偏倚。
4. 偏离预定干预偏倚。
5. 缺失数据偏倚。
6. 结局测量偏倚。
7. 选择性报告偏倚。

### 关键点

- 需要预设目标试验target trial。
- 混杂因素必须在protocol中定义。
- “严重偏倚”和“极严重偏倚”不应与RCT低风险研究直接等权合并。

## 3.3 ROBINS-E：非随机暴露研究

适用于暴露/危险因素/环境因素等非随机暴露研究。

### 关键点

- 核心是暴露分类、混杂、选择偏倚和结局测量。
- 暴露窗口和剂量测量要充分描述。
- 不能把所有观察性研究默认等同为“低质量”。

## 3.4 QUADAS-2：诊断准确性研究

### 四个领域

1. 患者选择。
2. Index test。
3. Reference standard。
4. Flow and timing。

每个领域评价偏倚风险；前三个领域还评价适用性 concerns。

### 常见高风险来源

- 病例-对照设计。
- 纳入已确诊病例和健康对照。
- 参考标准不独立或不准确。
- Index test解释时已知reference standard结果。
- 检测阈值事后选择。
- 并非所有患者接受同一参考标准。

## 3.5 PROBAST / PROBAST+AI：预测模型

适用于诊断或预后预测模型开发、验证、更新研究。

### 主要领域

1. Participants。
2. Predictors。
3. Outcome。
4. Analysis。

AI模型还应特别关注：

- 数据来源和代表性。
- 标签质量。
- 训练/验证/测试集划分。
- 数据泄漏。
- 超参数调优。
- 缺失值与类别不平衡。
- 外部验证。
- 校准度。
- 可解释性与部署场景。

## 3.6 AMSTAR 2：系统综述方法学质量

AMSTAR 2用于评价系统综述，尤其是干预效果综述。它有16个条目，不建议计算总分，整体可信度基于关键领域缺陷。

### 关键领域通常包括

- protocol是否预先制定。
- 检索是否充分。
- 排除研究清单和理由。
- 纳入研究偏倚风险评价。
- Meta分析方法是否合理。
- 发表偏倚评价。
- 利益冲突报告。

## 3.7 ROBIS：系统综述偏倚风险

ROBIS关注系统综述本身的偏倚风险，而不是原始研究。

### 适用场景

- 伞状综述。
- Overview of reviews。
- 评价现有综述可信度。

## 4. GRADE证据确定性

GRADE评价的是“某一结局的一组证据”的确定性，不是单篇研究质量。

### 4.1 起点评级

- RCT：通常从高确定性开始。
- 观察性研究：通常从低确定性开始。
- 但起点不是终点，需根据五个降级因素和三个升级因素判断。

### 4.2 五个降级因素

1. Risk of bias：纳入研究偏倚风险。
2. Inconsistency：结果不一致。
3. Indirectness：人群、干预、对照、结局或证据链间接。
4. Imprecision：样本量不足、CI过宽、跨越临床决策阈值。
5. Publication bias：发表偏倚或小样本效应。

### 4.3 三个升级因素

主要用于观察性证据：

1. 大效应。
2. 剂量反应关系。
3. 所有残余混杂都可能降低已观察效应或造成相反方向偏倚。

### 4.4 证据确定性表达

| 等级 | 含义 | 建议表述 |
|---|---|---|
| High | 非常确信真实效应接近估计值 | 明确支持/不支持 |
| Moderate | 真实效应可能接近估计值，但仍可能有重要差异 | 可能有效/可能有关 |
| Low | 对效应估计信心有限 | 效应不确定 |
| Very low | 对效应估计几乎没有信心 | 不能作可靠判断 |

## 5. 网状Meta证据评价

网状Meta除常规GRADE维度外，还要考虑：

- 传递性。
- 直接与间接证据一致性。
- 网络结构。
- 比较特异的证据确定性。
- 排名不确定性。

可使用CINeMA辅助评价NMA证据可信度。

## 6. 伞状综述证据可信度

伞状综述常用证据分级维度：

- 纳入综述方法学质量。
- 原始研究数量和样本量。
- 随机效应P值。
- 最大单项研究显著性。
- 异质性。
- 预测区间。
- 小样本效应。
- excess significance bias。
- 原始研究重叠程度。

## 7. 小样本效应与发表偏倚

### 7.1 何时做

一般建议研究数≥10时再使用漏斗图和Egger类检验。研究数较少时，图形和检验都不稳定。

### 7.2 如何解释

漏斗图不对称可能来自：

- 发表偏倚。
- 真实异质性。
- 小研究质量更差。
- 小样本效应。
- 结局测量差异。
- 统计偶然。

## 8. 选择性报告与缺失证据

### 8.1 常见来源

- 研究未发表。
- 结局未报告。
- 某时间点未报告。
- 亚组结果选择性报告。
- 研究只报告P值而不报告效应量。
- 注册结局与发表结局不一致。

### 8.2 应对策略

- 检索注册平台和protocol。
- 比对试验注册与发表论文。
- 联系作者。
- 使用ROB-ME或Cochrane missing evidence框架。
- 在GRADE中因发表偏倚/缺失结果降级。

## 9. 工具选择速查

| 课题类型 | 报告规范 | 偏倚风险 | 证据分级 |
|---|---|---|---|
| RCT干预Meta | PRISMA 2020 | RoB 2 | GRADE |
| 非随机干预Meta | PRISMA 2020 | ROBINS-I | GRADE |
| 危险因素Meta | PRISMA 2020/MOOSE辅助 | ROBINS-E/NOS/JBI | GRADE或领域框架 |
| 患病率Meta | PRISMA 2020 | JBI/AXIS | GRADE可适配 |
| 诊断Meta | PRISMA-DTA | QUADAS-2 | GRADE-DTA |
| 网状Meta | PRISMA-NMA | RoB 2/ROBINS-I | CINeMA/GRADE-NMA |
| 预测模型Meta | TRIPOD-SRMA | PROBAST/PROBAST+AI | 模型证据框架 |
| 伞状综述 | PRISMA 2020 + overview规范 | AMSTAR 2/ROBIS | 证据可信度分级 |
| IPD-MA | PRISMA-IPD | 原研究RoB + IPD完整性 | GRADE |

## 10. 顶刊审稿红线

1. 用Jadad或NOS总分决定纳入/排除。
2. 没有protocol或protocol与论文不一致且未解释。
3. 偏倚风险评价没有双人独立完成。
4. 用PRISMA评分评价纳入研究质量。
5. 只报告“高质量/低质量”而不说明偏倚领域。
6. 证据确定性分级与结论措辞不一致。
7. 对高异质性只说“采用随机效应模型解决”。
8. 对发表偏倚只做漏斗图，不讨论选择性报告和灰色文献。
